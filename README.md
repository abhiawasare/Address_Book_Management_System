# Address Book Management System

This is a simple console-based Address Book project developed in C language.
It helps users store and manage contact details like name and phone number.

## Features
- Add new contact
- Edit contact
- Delete contact
- Search contact
- File handling to save data

## Technologies Used
- C Programming
- Structures
- File Handling
- GCC Compiler

## How to Run
gcc *.c
./a.out
